# 太极-八卦-DNA：基于道家哲学与DNA四进制的分子中文计算系统

## 📖 项目简介
本项目尝试以道家“太极→两仪→四象→八卦→六十四卦→万物”的生成论为顶层架构，以DNA的四种碱基（A/T/C/G）为物理基底，构建一个能够模拟汉字自组装的分子计算系统。

这不是一个能运行的软件，而是一个**概念验证与模拟设计**项目。所有设计均可在免费软件NUPACK中进行模拟验证。

## 🧬 核心思想
| 道家层次 | 物理实现（DNA） | 计算功能 |
|----------|----------------|----------|
| 太极 | DNA反应溶液 | 计算发生的“场” |
| 两仪 | 嘌呤/嘧啶 | 二进制底层 |
| 四象 | A/T/C/G | 四进制数据单元 |
| 八卦 | 二碱基组合 | 8个基础字根 |
| 六十四卦 | 三碱基序列 | 常用汉字编码 |
| 万物 | 长链DNA | 句子/知识表达 |
